本框架的国际化相关配置都存储到数据库中，而不是message.properties。国际化的key默认使用中文。

**国际化语言的切换**

1、html上的使用：

```html
[[#{xxx}]]  其中xxx表示国际化的key，例如[[#{立即登录}]]
```

**使用方式**

1、html上的使用：

```html
[[#{xxx}]]  其中xxx表示国际化的key，例如[[#{立即登录}]]
```

2、java上的使用：

```html
I18nUtils.t("xxx"); //其中xxx表示国际化的key，例如I18nUtils.t("立即登录");
```
